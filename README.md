# The Ultimate Modpack
An untterly gigantic conjunction of mods expanding the number of resources, units, buildings, and eras. Mods have been welded and expanded upon.

Credited Mods:
    - Warfare Expanded - General Wadaling/The Bucketeer
    - Extra Resources - Cavenir 
    - Arial expanded - Huymnsel 
    - Future Expansion - Sullien 
    - Horses and Iron - kingkarna1 
    
    What is the image? 
    
    This mod is intended for the slow-paced strategic player and is mainly designed to be cool as opposed to balanced. While it's not intended to be wretchedly unbalanced, it is tilted when it comes to access to resources. A player in a warm region will outclass a tundra region player in the early game; however, the wheel of fortune continuously turns.
